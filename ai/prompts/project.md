1. 專題名稱 / Project Title / 프로젝트명
   校園社團網站

2. 專案目的 / Purpose / 목적
   強化社團內部營運與管理。提供一套便利的工具，以簡化社團的日常管理事務（如活動發布、成員溝通、資料交接），旨在解決「社團幹部管理混亂、交接不易」的核心問題。

3. 主要使用者 / Target Users / 주요 사용자
   - 社團幹部 (社長、活動、公關等)：網站的核心管理者，負責發布資訊、管理活動與成員。
   - 一般社團成員：網站的主要內容消費者，用以接收資訊、報名活動。

4. 網站主要功能 / Key Features / 주요 기능
   - 成員管理系統：幹部可管理成員名單、職位與權限。
   - 公告系統：幹部可對內發布最新消息。
   - 活動日曆與報名：幹部可建立活動，成員可查看與報名。
   - 文件共享區：幹部可上傳文件（如會議記錄、規章），方便查閱與傳承。

5. 成功指標 / Success Metrics / 성공 지표
   - 社團幹部採用率：上線一學期後，超過 80% 的社團幹部至少使用過一次管理功能。
   - 平台活動建立數：平均每月每個社團透過平台建立 2 個以上活動。
   - 成員活躍度：每月有 60% 的社團成員至少登入一次。
   - 幹部主觀回饋：透過問卷，超過 70% 的幹部認為平台讓管理工作變得更輕鬆。

6. 非範圍內容 / Out of Scope / 범위 밖
   - 線上金流功能 (Online Payment)
   - 即時通訊功能 (Real-time Chat)
   - 對外招生的公開頁面 (Public-facing Recruitment Pages)
   - 複雜的影音串流與相簿功能 (Complex Media Streaming/Albums)
